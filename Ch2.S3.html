2.3 Steady states and their stability: single population

Special values of NN are those where f⁢(N)f(N) is actually equal to 0. We say there is a steady state at N=N∗N=N^{*} if f⁢(N∗)=0f(N^{*})=0. Usually, there is more than one such N∗N^{*}.

We note that f⁢(N∗)=0f(N^{*})=0 means that, if a population starts at size N∗N^{*}, it will stay at size N∗N^{*}. But, what if the population starts close to N∗N^{*}? Roughly speaking, we say that the steady state N∗N^{*} is “stable” if after a small perturbation in the system (i.e., it moves from N∗N^{*} somewhere very close), the population goes back to N∗N^{*}. On the other hand, we say that the steady state is “unstable” if after a small perturbation, the system moves away from the steady state. Graphically, we can see that:
