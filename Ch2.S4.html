2.4 Harvesting

Imagine that a population (this time a good example would be a population of fish) obeys the logistic equation. Now imagine that we want to harvest some of the population (in this case, catch some fish). What is the optimal rate of harvesting and how does harvesting affect the population?

We include harvesting in the model by modifying (2.1) to

d⁢N⁢(t)d⁢t=r⁢N⁢(t)⁢(1−N⁢(t)K)−α⁢N⁢(t)=r⁢N⁢(t)⁢(1−αr−N⁢(t)K).\frac{{\rm d}N(t)}{{\rm d}t}=rN(t)\left(1-\frac{N(t)}{K}\right)-\alpha N(t)\ =% \ rN(t)\left(1-\frac{\alpha}{r}-\frac{N(t)}{K}\right). (2.5)

The extra, harvesting, term is −α⁢N-\alpha N. (That is, the number of fish caught at any time is proportional to the number that are there.) Notice that [α]=T−1[\alpha]=T^{-1}.

    If α>r\alpha>r then f⁢(N)<0f(N)<0 and the population size always decreases. In the long run there are no fish! We can say that overfishing causes extinction.

    If α<r\alpha<r then there is a stable steady state. The size of the population in the long-term reaches N∗=r−αr⁢KN^{*}=\frac{r-\alpha}{r}K, which means that the steady-state rate of harvesting is

    α⁢N∗=α⁢r−αr⁢K.\alpha N^{*}=\alpha\frac{r-\alpha}{r}K.

What is the optimum value of α\alpha (if we want to sustainably catch as many fish as possible)? Let us define the steady-state rate of harvesting as a function of α\alpha:

h⁢(α)=α⁢r−αr⁢K.h(\alpha)=\alpha\frac{r-\alpha}{r}K.

To find the maximum, take h′⁢(α)=K−2⁢Kr⁢αh^{\prime}(\alpha)=K-\frac{2K}{r}\alpha. We conclude that the optimum value of alpha is αmax=r2\alpha_{\rm max}=\frac{r}{2}; see Figure 2.8.

Does this all make sense? What is α\alpha? We know that α\alpha and rr are rates with dimensions T−1T^{-1}. The way to sustainably harvest the maximum from the population (to catch the most fish you can without driving the population to extinction) is to choose α\alpha to be half of rr. In that case, the steady state size of the fish population will be half as big as if there were no fishing at all.
